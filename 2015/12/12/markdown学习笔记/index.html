Markdown 的语法的简要规则

1.标题

标题是每篇文章都需要也是最常用的格式，在Markdown中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

总共六级标题，建议在 # 号后面加一个空格，这是最标准的 Markdown 语法。

2.列表

熟悉 HTML 的同学肯定知道有序列表与无序列表的区别，在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格。

3.引用

只需要在文本前加入 > 这种尖括号即可。

> 你的内容

4.图片与链接

插入链接与图片的语法很像，区别在一个 ！ 号

图片为：！[](){ImgCap}{/ImgCap}

链接为： []()

5.粗体与斜体

Markdown的粗体和斜体也非常简单，用两个 *包含一段文本就是粗体的语法，用一个 *包含一段文本就是斜体的语法。

例如：我是粗体 我是斜体

6.表格

表格的比较麻烦，例子如下：

| 姓名        | 年龄     |  得分 |
| ---------- | :------:|------:|
| 小张        |26       |98     |
|小李         |34       |95     |

效果如下

姓名 年龄 得分
小张 26 98
小李 34 95

7.代码框

如果你是个程序员，需要在文章里优雅的引用代码框，在Markdown下也非常简单。

单行代码框

单行代码框可以显示比较简短的代码，通常不超过一行用两个 ` 把中间的代码包裹起来就可以了

多行代码框

多行代码框有两种写法第一种是使用 tab 键把每一行代码缩进，或者打4个空格

第二种写法是第一行和最后一行写三个```中间行是代码

8.分割线

分割线的语法只需要三个***
例如:

#Markdown
